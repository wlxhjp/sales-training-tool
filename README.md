# 🖥️ 租赁电子设备销售培训 · 多维出题工具

> 基于 **DeepSeek** 大模型的智能销售培训辅助工具 —— 输入一道题目，从 6 个专业维度自动生成衍生培训题目。

## ✨ 功能特点

- 🎯 **客户痛点维度**：引导学员思考客户的成本压力、设备贬值、IT负担等核心痛点
- 💻 **产品特性维度**：从性能参数、配置灵活度、品牌选择等角度重新审视题目
- 🏢 **使用场景维度**：结合展会、短期项目、远程办公等具体业务场景
- 🛡️ **异议处理维度**：预判并应对"租不如买""二手不安全"等常见异议
- ⚖️ **竞品对比维度**：租赁 vs 购买、不同服务商之间的优劣势分析
- ✨ **价值主张维度**：提炼降本增效、CAPEX转OPEX、一站式服务等核心价值

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的 DeepSeek API Key
# 获取地址：https://platform.deepseek.com/api_keys
```

### 3. 运行程序

```bash
python main.py
```

## 📁 项目结构

```
sales-training-tool/
├── .env                 # 环境变量（API Key，不上传 Git）
├── .env.example         # 环境变量模板
├── .gitignore           # Git 忽略规则
├── requirements.txt     # Python 依赖
├── README.md            # 项目说明文档
└── main.py              # 主程序入口
```

## 🔧 技术栈

| 组件 | 说明 |
|------|------|
| [DeepSeek](https://deepseek.com) | 大语言模型（deepseek-chat） |
| [OpenAI Python SDK](https://github.com/openai/openai-python) | 兼容 DeepSeek API 格式 |
| python-dotenv | 环境变量管理 |

## 📝 自定义扩展

如需新增思考维度，只需在 `main.py` 的 `DIMENSIONS` 字典中添加新的条目：

```python
"7": {
    "name": "📊 你的新维度",
    "prompt": "你的 System Prompt...",
},
```
